# MPK Mebel Landing

Обновлённая версия лендинга для `mpk-mebel.ru`, собранная как один статический `index.html`.

## Что улучшено

- Убрана визуальная перегруженность и повторяющиеся блоки.
- Усилен первый экран: ценностное предложение, доверие и CTA видны сразу.
- Пересобрана структура в логичный путь: оффер -> портфолио -> преимущества -> процесс -> акции -> отзывы -> FAQ -> контакты.
- Контакты и заявка упрощены: форма собирает сообщение и отправляет его в WhatsApp.
- Сайт стал лучше читать на мобильных устройствах и проще поддерживать.

## Структура файла

- `header`: навигация и быстрые контакты.
- `hero`: главный оффер и ключевые преимущества.
- `projects`: визуальная витрина направлений и проектов.
- `advantages`: основные коммерческие причины выбрать компанию.
- `process`: понятный путь работы.
- `story`: производство, команда и гарантия.
- `offers`: специальные предложения.
- `reviews`: упрощённый блок доверия с переходом на отзывы.
- `faq`: ответы на частые вопросы.
- `contacts`: контакты и форма заявки.

## Как редактировать

- Основной контент находится прямо в `index.html`.
- Для изменения контактов обновите:
  - ссылки `tel:`
  - ссылки `wa.me`
  - ссылку на Telegram
  - адрес и email в блоке контактов
- Для замены фото достаточно изменить URL в секции `#projects`.
- Для текста кнопки и поведения формы смотрите блок `script` в конце файла.

В репозитории уже настроен GitHub Actions деплой на сервер.

- Текущий deploy настроен через пользователя `deploy` и SSH-ключ.
